# Architecture - Memory Guard

## 🧱 技术栈

- Rust
- Tauri 2
- React + TypeScript

---

## 🧩 模块划分

### monitor

- 获取系统内存
- 按需获取进程列表

### alert

- 阈值判断
- 冷却控制
- 通知触发

### process_control

- 退出应用
- 强制退出

### tray

- 菜单栏状态
- 面板控制

### settings

- 本地配置管理

---

## ⚙️ 核心设计

### 分层监控

#### 后台模式

- 只读取总内存
- 不扫描进程
- 每 2 秒执行

#### 面板模式

- 扫描进程
- 排序
- 展示数据

#### 超阈值

- 仅触发一次进程扫描
- 获取 top 应用

---

## 🔁 刷新策略

| 状态     | 行为          |
| -------- | ------------- |
| 后台     | 2s 内存检测   |
| 面板打开 | 2~3s 扫描进程 |
| 高压     | 降级到 4~5s   |

---

## ⚡ 性能约束

- 内存占用 < 50MB
- CPU 空闲 < 1%
- 后台禁止全量扫描

---

## 🧵 并发模型

- 后台线程：监控
- UI线程：渲染
- 不允许阻塞 UI

---

## 🗄️ 数据策略

- 不保存长期历史
- 仅缓存最近数据（<100条）

---

## 🔐 安全约束

- 禁止关闭自身
- 禁止关闭关键系统进程
- 默认使用安全退出（terminate）
